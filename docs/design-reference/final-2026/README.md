## Implementation Rules

1. **Device frames in mockup images are presentation ONLY and must NOT be rendered in the actual app UI**
2. Copy the internal app screen design only
3. Preserve RTL alignment
4. Preserve routing
5. Preserve app functionality
6. Do not add iPhone hardware frames, status bars, dynamic island, or device mockup borders
7. Match layout, typography, colors, spacing, cards, icons, and ornaments exactly

## Design Language: Saudi Premium Minimal

- Warm ivory background (#FAF7F2)
- Soft sand texture with muted gold accents (#A78042)
- Premium Arabic typography with Noto Sans Arabic
- Rounded cards with subtle shadows
- Thin gold dividers and ornamental separators
- Palm/desert/lantern motifs
- Soft low-contrast background imagery
- Clear RTL alignment
- Balanced text hierarchy
