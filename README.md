# 🟦 مواعيدك (Mawaeedak) v3.0.0

Production-ready Saudi Appointments System

## Quick Start

```bash
cd mawaeedak_api && npm install && cp .env.example .env && npm start
```

## Security

- JWT Authentication
- bcrypt Hashing
- Rate Limiting
- CORS Protection

## Production Checklist

- Set JWT_SECRET
- Set ADMIN credentials
- Configure CORS_ORIGIN
