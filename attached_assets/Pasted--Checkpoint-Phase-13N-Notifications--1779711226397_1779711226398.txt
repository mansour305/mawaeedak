2. المطلوب في Admin Login / Dashboard
==================================================

ارفع Admin Login/Dashboard من 7.5/10 إلى 9/10 أو أعلى.

المطلوب:
1. صفحة تسجيل الدخول يجب أن تتبع نفس الهوية:
   - خلفية ورقية/تراثية.
   - بطاقة login كريمية أو بنية داكنة فاخرة.
   - شعار مواعيدك واضح.
   - عبارة "لوحة المالك" أو "تسجيل دخول الإدارة" بشكل فاخر.
   - أزرار بنية/ذهبية.
   - حقول input متناسقة.
   - لا مظهر Supabase/default خام.

2. Admin Dashboard بعد الدخول يجب أن يكون متناسقاً:
   - sidebar بني داكن.
   - شعار مواعيدك واضح.
   - بطاقات إحصائية كريمية/ذهبية.
   - أزرار وأيقونات ذهبية.
   - tables/cards بنفس نظام التصميم.
   - لا admin default منفصل عن التطبيق.

3. لا تغيّر منطق الحماية:
