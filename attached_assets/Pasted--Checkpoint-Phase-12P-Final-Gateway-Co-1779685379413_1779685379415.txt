4. تدقيق Orval المتبقي
==================================================

ابحث عن كل استخدامات Orval hooks أو API clients.

صنّفها:
1. Mutations ما زالت API عمداً.
2. Endpoints محسوبة server-side ولا يوجد بديل Supabase.
3. Auth/admin لا علاقة لها بـ Orval.
4. بقايا قديمة يجب إصلاحها.

مهم جداً:
- إذا وجدت NotificationsPage لا تزال تستخدم Orval للقراءة أو mark-read أو delete، أصلحها أو اذكر أن التقرير السابق خطأ.
- إذا وجدت TopBar count لا يزال Orval، أصلحه أو اذكر السبب.
- إذا وجدت AdminNotifications read/delete لا يزال Orval رغم Phase 12K، أصلحه أو اذكر السبب.
- لا تحول send notification إذا بقي API عمداً بسبب fan-out.
