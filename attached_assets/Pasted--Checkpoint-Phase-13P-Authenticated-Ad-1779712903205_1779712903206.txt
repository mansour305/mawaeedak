==================================================
1. المطلوب
==================================================

أكمل Screenshot Package الناقص فقط.

يجب تجهيز صور Preview فعلية لهذه الصفحات:

1. Smart Calculators
- افتح /finance
- اضغط تبويب "الحاسبات"
- خذ Screenshot واضح للتبويب بعد الفتح
- يجب أن يظهر تصميم الحاسبات الفعلي، وليس فقط زر التبويب

2. Admin Dashboard
- سجّل الدخول في /admin بحساب المالك
- افتح Dashboard بعد الدخول
- خذ Screenshot واضح
- يجب أن يظهر:
  - sidebar
  - شعار مواعيدك
  - stat cards
  - activity list
  - الهوية التراثية

3. Admin Automation
- بعد تسجيل الدخول
- افتح /admin/automation
- خذ Screenshot واضح
- يجب أن يظهر:
  - daily automation
  - notification scheduler
  - logs/status
  - timezone note
  - الهوية التراثية

4. Admin Visual Guide
- بعد تسجيل الدخول
- افتح /admin/visual-guide
- خذ Screenshot واضح
- يجب أن يظهر:
  - colors
  - typography
  - cards
  - buttons
  - icons
  - logo/emblem
  - visual tokens

5. Story Share / Daily Generated Card
- افتح StoryPage
- اعرض بطاقة المشاركة/التصدير أو preview اليومي
- تأكد أن البطاقة تحتوي:
  - شعار مواعيدك
  - عبارة "مواعيدك"
  - ويفضل "مواعيدك — جميع الحقوق محفوظة"
- إذا النص الكامل غير موجود، أضفه في بطاقة المشاركة/الصورة اليومية فقط، بدون إزعاج واجهة المستخدم.
