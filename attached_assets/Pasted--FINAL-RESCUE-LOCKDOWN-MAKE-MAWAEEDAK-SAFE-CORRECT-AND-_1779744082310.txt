2. Fix Public Admin Leak
==================================================

أصلح فوراً:
زر "لوحة التحكم" لا يظهر إلا إذا:
- session موجودة
- role = admin أو super_admin

الحالات:
anonymous → لا يرى لوحة التحكم
role=user → لا يرى لوحة التحكم
admin/super_admin → يرى لوحة التحكم

كل /admin/* محمي:
anonymous → login
user → access denied
admin/super_admin → dashboard

اختبر الحالات الثلاث.
